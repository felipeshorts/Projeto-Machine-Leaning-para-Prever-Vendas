# Projeto-Machine-Leaning-para-Prever-Vendas
Pequeno projeto criado no Azure para prever melhores datas de vendas de sorvetes.

Neste projeto, pude aprender com experimentos de previsão de vendas utilizando Azure Machine Learning, a fim de obter dados e estatísticas com o objetivo de identificar melhores datas e períodos climáticos para potencializar as vendas de sorvete.
